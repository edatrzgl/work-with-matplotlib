# Matplotlib Data Visualization Examples

This repository contains a Jupyter Notebook demonstrating various data visualization techniques using **Matplotlib** with real-world datasets.

The notebook showcases:
- Line plots
- Horizontal bar charts
- Pie charts
- Regular bar charts
- Subplots layout for multiple visualizations

## Notebook Overview

**File:** `Matplotlib.ipynb`

### Visualizations Included:
1. **Monthly Sales Trend** → Line chart with markers
2. **Accessory Category Sales** → Horizontal bar chart
3. **Bike Category Sales Distribution** → Pie chart with percentage labels
4. **Sales by Country** → Vertical bar chart

All charts are combined in a clean 2×2 subplot layout.

## Datasets Used

The notebook works with the following CSV files (located in `Python Course/Data for Python Course/`):

- `web_sales.csv` → Main sales data (used for most visualizations)
- `2018-2019 Happiness.csv` → World Happiness Report
- `airbnb.csv` → Airbnb listings
- `salaries.csv` → Salary information
- `Ethereum Historical Data.csv` → Ethereum price history

**Note:** The actual data files are **not** included in this repository.  
You need to place them in the correct folder structure or update the file paths in the notebook.
